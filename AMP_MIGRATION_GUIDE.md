**Problem:** Limited CSS size (75KB)  
**Solution:** Optimize CSS, use AMP components

## 📞 Troubleshooting

### Ads Not Showing?
1. Check AMP validation (add #development=1)
2. Verify Publisher ID is correct
3. Check AdSense account status
4. Wait 24 hours for indexing
5. Clear browser cache

### Page Not Loading?
1. Check browser console for errors
2. Verify all AMP components are used correctly
3. Check for non-AMP JavaScript
4. Validate HTML structure
5. Test on different device

### Performance Issues?
1. Check file sizes
2. Optimize images
3. Minimize CSS/HTML
4. Use AMP caching (automatically done)
5. Test on slow network
